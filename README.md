# ManiLocal

Research and planning for running local AI models, focused on content management workflows.

## What This Is

A structured research repo for evaluating and operationalizing self-hosted AI models for content management tasks — drafting, editing, summarization, classification, and knowledge base maintenance.

## Goals

- **Evaluate** local model runtimes (Ollama, llama.cpp, vLLM, LM Studio)
- **Benchmark** models on content management tasks
- **Document** hardware requirements and performance tradeoffs
- **Build** reproducible local inference pipelines for content workflows

## Structure

| Directory | Purpose |
|-----------|---------|
| `research/` | Model evaluations, benchmarks, comparison notes |
| `configs/` | Runtime and model configuration files |
| `scripts/` | Setup, benchmarking, and utility scripts |
| `pipelines/` | Content management pipeline definitions |
| `docs/` | Extended documentation |
